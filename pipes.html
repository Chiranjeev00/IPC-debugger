How Pipes Work

📝 Write Operation

Process A calls write() to send data into the pipe buffer. If buffer is full in blocking mode, write blocks until space is available.

📖 Read Operation

Process B calls read() to retrieve data from the pipe. If buffer is empty in blocking mode, read blocks until data arrives.

⚡ Buffer Behavior

Pipe uses a fixed-size buffer (typically 4KB). Data flows in FIFO order. Non-blocking mode returns immediately with error if buffer is full/empty.
